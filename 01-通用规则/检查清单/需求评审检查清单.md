# 需求评审检查清单

## 评审前（自检）
- [ ] PRD 已完成并通过自审
- [ ] 涉及的数据模型已梳理
- [ ] 异常处理逻辑已覆盖
- [ ] 原型已同步更新
- [ ] 旧功能兼容性已评估

## 评审中
- [ ] 业务目标是否清晰
- [ ] 用户场景是否完整
- [ ] 功能边界是否明确（含/不含）
- [ ] 与现有功能的交互是否合理
- [ ] 数据口径是否与后端对齐
- [ ] 权限逻辑是否明确
- [ ] 异常流程是否列出

## 评审后
- [ ] 评审意见已记录
- [ ] 修改项已明确责任人和截止日
- [ ] PRD 已更新并同步相关方

## 上线前
- [ ] 功能测试通过
- [ ] 数据验证通过
- [ ] 用户手册已更新
- [ ] 培训/演示材料已准备
