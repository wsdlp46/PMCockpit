# 01-通用规则

> 项目无关的通用资产，跨项目复用。新项目启动时优先从这里取用。

## 目录

| 目录/文件 | 内容 | 核心文件 |

| **项目工作流规范.md** | 四层结构、工作流阶段、双层知识库治理、AI 行为契约（合并了原知识库搭建指南） | 项目工作流规范.md |
| **VitePress文档站搭建与维护.md** | 文档站搭建部署、日常维护、已知坑 | VitePress文档站搭建与维护.md |

| **设计规范/** | 引用自外部的固定设计标准，不可修改参数 | [政务APP设计规范.md](设计规范/政务APP设计规范.md)、[Web应用设计规范.md](设计规范/Web应用设计规范.md) |

| **模板/** | 文档和项目管理模板 | [PRD模板.md](模板/PRD模板.md)、[系统上线项目管理模板.md](模板/系统上线项目管理模板.md)、[可研方案-参考_转换.md](模板/可研方案-参考_转换.md) |

| **术语表.md** | 行业/公司/项目术语速查 | 术语表.md |

## 使用场景

| 场景 | 取用什么 |
|------|----------|
| 新项目启动 | PRD模板 → 需求评审检查清单 |
| 画原型 | 设计库基线 → 设计规范（组件参数）→ 方法论 |
| UI走查/评审 | 设计检查清单 |
| 写文档遇到术语 | 术语表 |
| 项目上线排期 | 模板 → 系统上线项目管理模板 |
| 查设计约定 | 设计库 → 设计约定.md |
| 查历史案例 | 03知识库/工作案例库/ |
| 搭建/维护文档站 | VitePress文档站搭建与维护.md |
