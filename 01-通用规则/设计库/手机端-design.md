# 手机端 设计基线

> 适用于：H5轻应用、微信小程序、移动端页面
> 默认主色 #3079ED（政务APP蓝），可被项目自定义色覆盖
> 画板基准：375px宽（iPhone一倍图），高度按内容流
> 参照：政务APP设计规范 + <项目名>H5专区

---

## 一、画框与布局

| 参数 | 值 |
|------|-----|
| 手机模拟框宽度 | 375px |
| 模拟框高度 | 850px（内容超长可滚动） |
| 模拟框圆角 | 8px |
| 模拟框阴影 | 0 4px 24px rgba(0,0,0,0.12) |
| 页面背景 | #FFFFFF |
| 内容区内边距 | 左右16px（安全边距） |
| 内容区布局 | 单列流式，垂直滚动 |

---

## 二、顶部区域

### 2.1 状态栏

| 参数 | 值 |
|------|-----|
| 高度 | 28px |
| 背景 | 与导航栏同色（主题色渐变） |
| 文字色 | rgba(255,255,255,0.85) |
| 字号 | 11px |
| 布局 | 左右两端（时间 + 信号/WiFi/电量图标） |
| 内边距 | 左右20px，顶部8px |

### 2.2 导航栏

| 参数 | 值 |
|------|-----|
| 高度 | 44px |
| 背景 | 主题色渐变（如 #C53030→#E74C3C） |
| 返回箭头 | 左侧16px处, 16px, 白色, cursor pointer |
| 标题 | 居中, 17px, font-weight 500, 白色 |
| 右侧占位 | 16px宽空白（保持标题居中） |

---

## 三、Banner（顶部横幅）

| 参数 | 值 |
|------|-----|
| 高度 | 160-200px |
| 背景 | 主题色渐变（深→浅，多层过渡） |
| 底部过渡 | `::after` 白色弧形（border-radius 50% 50% 0 0, bottom:-30px, height:80px） |
| 标题 | 左下角20px处, 22px, 500, 强调色（如 #FFE066） |
| 副标题 | 标题下方, 13px, rgba(255,255,255,0.7) |
| 装饰 | 右侧SVG五角星/几何图形, opacity 0.15-0.35 |

---

## 四、浮动卡片（叠Banner弧）

当首页Banner下方需要卡片时，使用浮动叠加：

| 参数 | 值 |
|------|-----|
| 容器 | margin-top -60px, position relative, z-index 10 |
| 左右边距 | 12px |
| 卡片背景 | #FFFFFF |
| 卡片圆角 | 12px |
| 卡片阴影 | 0 4px 20px rgba(0,0,0,0.08) |
| 卡片内边距 | 16px |
| 装饰光晕 | `::before` 径向渐变（rgba(主色,0.06), transparent 70%）, 右上角60px |

## 五、列表卡片

### 5.1 左图标 + 右信息型（场所列表）

| 标签 | 11px, bg #FDF2F2, color 主题色, padding 2px 8px, 圆角4px |

## 九、色值（默认）

默认使用政务APP主色 #3079ED，项目可覆盖。

| 用途 | 色值 |
|------|------|
| 主色 | #3079ED（可覆盖为项目色） |
| Banner深 | 主色深15-20%（如 #C53030 对应 #B71C1C） |
| Banner中 | 主色（如 #D32F2F） |
| Banner浅 | 主色浅（如 #E53935） |
| 标题文字 | #333333 |
| 正文文字 | #606266 |
| 辅助文字 | #909399 |
| 浅色底 | #F5F7FA / #FDF2F2 / #FFF5F5 |
| 成功/生效中 | #009965, bg #E6F7EE |
| 已过期 | #E07000, bg #FFF3E0 |
| 未生效/禁用 | #909399, bg #F5F5F5 |
| 分隔线 | #F5F7FA |

## 十一、右侧注释面板（原型专用）

手机端原型同样使用B端原型的右侧注释面板模式，但面板内容偏重移动端特色：

| 参数 | 同web后台 |
|------|:---:|
| 面板宽度/背景/边框/内边距 | 同 web后台-design.md 第九节 |
| 内容差异 | 注释聚焦H5特征：Banner渐变、卡片浮叠、触控尺寸、按钮语义 |
