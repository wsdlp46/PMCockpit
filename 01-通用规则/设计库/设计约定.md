# 设计约定

> 跨项目共享的全局设计约束。项目级设计约定（如品牌主色、特定组件规范）存放在各项目的 `.workbuddy/memory/MEMORY.md` 中。

_最后更新：2026-06-18_

---

## 一、基础约定

### 单位体系

- 布局间距以 4px 为基准单位
- 原型画板宽度：web后台 1440px / web门户 1200px / 手机端 375px
- 桌面端默认字号 14px，移动端 15px

### 圆角规范

| 元素 | 圆角 |
|------|------|
| 卡片/面板 | 8px |
| 输入框/下拉框 | 6px |
| 按钮 | 6px |
| 标签/徽章 | 4px |
| 弹窗 | 12px |

### 阴影规范

| 层级 | 阴影值 | 用途 |
|------|--------|------|
| L1 卡片 | `0 1px 4px rgba(0,0,0,0.08)` | 卡片、面板 |
| L2 悬浮 | `0 4px 12px rgba(0,0,0,0.12)` | 表头固定、悬浮面板 |
| L3 弹窗 | `0 8px 24px rgba(0,0,0,0.16)` | 模态弹窗 |

### 原型文件结构

- 每个页面独立一个 HTML 文件
- 文件命名：`{模块缩写}{序号}-{页面名}-原型.html`
- 后台管理系统页面数 ≥ 2 时，需检查侧边栏菜单统一性

## 二、项目级设计约定（模板）

以下模板用于在项目 `.workbuddy/memory/MEMORY.md` 中建立项目专属设计约定：

```markdown
## 设计约定

### 品牌信息
- 项目名：XXX
- 品牌主色：#XXXXXX
- 辅助色：#XXXXXX / #XXXXXX

### 页面类型
- 后台管理：标准列表+表单 / ... 
- 门户：卡片矩阵 / 全宽大图 / ...
- H5：单列流式 / 卡片列表 / ...

### 组件规范
- 表格：带分页 / 虚拟滚动
- 弹窗：L1侧滑窗 / L2模态弹窗
- 按钮：主按钮 #XXX / 次按钮 #XXX
- 标签/状态：类型A=#XXX / 类型B=#XXX

### 原型文件结构
- [模块A]: 页面编号 A1-A5
- [模块B]: 页面编号 B1-B10
```
