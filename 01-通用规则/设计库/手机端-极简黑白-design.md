# 手机端 极简黑白 设计基线

> 适用于：移动端电商、钱包/支付、订单管理、个人中心类 App（H5 / 小程序 / 原生）
> 默认主色 #000000（纯黑），背景 #FFFFFF，无主题色渐变
> 画板基准：375px宽（iPhone一倍图），高度按内容流
> 风格参照：My Orders / E-Wallet / Profile 系列原型（黑实心按钮 + 白卡片 + 中圆角）

---

## 〇、风格定位

| 维度 | 取值 |
|------|------|
| 主题 | 浅色（Light），白底黑字 |
| 主操作色 | 纯黑 #000000（非品牌色，非渐变） |
| 视觉密度 | 宽松留白，卡片间距大 |
| 圆角语言 | 中等圆角 8–12px，按钮偶用胶囊 |
| 阴影 | 极浅或无，靠分隔线与留白分层 |
| 图标 | 线性（stroke 1.5–2px），与文字同色 |
| 装饰 | 几乎无，强调内容本身 |

> 与 `手机端-design.md`（政务APP政务风：渐变 Banner + 浮叠卡片 + 强调色）的区别：本基线**去渐变、去强调色、去装饰光晕**，靠黑白对比与排版建立层级。

## 十八、与项目主色的协作

本基线默认主操作色为 **#000000**。若项目要求品牌色介入，按以下优先级覆盖：

```
1. 按钮主色（实心）         → 可改为品牌色，但保持实心无渐变
2. 选中态边框/单选实心       → 跟随按钮主色
3. 状态文字（成功/警告/危险）→ 维持原语义色，不跟随品牌色
4. 其余（背景/文字/分隔）    → 不变
```

> 改色原则：**只换"操作色"这一根线，不动黑白骨架**。这样既融入品牌，又不破坏极简基调。
