# web门户 设计基线

> 适用于：对外门户首页、聚合页、产品介绍页、数据资产门户
> 默认主色 #1E63B2，可被项目自定义色覆盖
> 区别于web后台：全宽无侧边栏、大图Banner、卡片矩阵、偏营销展示

---

## 一、布局

| 参数 | 值 |
|------|-----|
| 内容最大宽度 | 1200px |
| 水平居中 | margin 0 auto |
| 页面背景 | #F8F9FA 或 #FFFFFF |
| 节（section）间距 | padding 60-80px |
| 节交替背景 | 白/浅灰交替（视觉节奏） |
| 响应式 | ≥768px 正常布局，<768px 单列堆叠 |

---

## 二、色值

| 用途 | 色值 |
|------|------|
| 主色 | #1E63B2（默认，可覆盖） |
| 主色渐变 | linear-gradient(135deg, #1E63B2, #3B82F6) |
| 标题文字 | #1A1A2E 或 #1F1F1F |
| 正文文字 | #4B5563 |
| 辅助文字 | #9CA3AF |
| 白色卡片背景 | #FFFFFF |
| 卡片阴影 | 0 2px 12px rgba(0,0,0,0.08) |
| 深色底文字 | #FFFFFF |

---

## 三、字体

| 用途 | 字号 | 字重 |
|------|:---:|:---:|
| Hero大标题 | 36-48px | 700 |
| Hero副标题 | 16-18px | 400 |
| 节标题 | 28-32px | 600 |
| 卡片标题 | 18px | 600 |
| 卡片正文 | 14px | 400 |
| 统计数字 | 32-40px | 700 |
| 统计标签 | 14px | 400 |
| CTA按钮 | 16px | 500 |

---

## 四、Hero Banner（首屏）

| 参数 | 值 |
|------|-----|
| 高度 | 400-600px |
| 背景 | 主色渐变 + 装饰几何元素 |
| 内容布局 | 居中，标题→副标题→CTA按钮（纵向排列） |
| 标题间距 | margin-bottom 16px |
| 副标题间距 | margin-bottom 32px |
| CTA按钮 | 主色圆角大按钮，padding 12px 32px, 16px, 500 |
| 滚动提示 | 底部箭头或"了解更多"（选配） |

---

## 五、统计卡片区

| 参数 | 值 |
|------|------|
| 布局 | flex 或 grid，3-4列 |
| 卡片背景 | #FFFFFF |
| 卡片阴影 | 0 2px 12px rgba(0,0,0,0.08) |
| 卡片圆角 | 12px |
| 卡片内边距 | 24px |
| 数字样式 | 32-40px, 700, 主色 |
| 标签样式 | 14px, 400, #6B7280, margin-top 8px |
| 图标点缀 | 40-48px色块圆圈 + 图标, margin-bottom 16px |

---

## 六、内容卡片区

| 参数 | 值 |
|------|------|
| 列数 | 3列 grid（gap 24px） |
| 卡片背景 | #FFFFFF |
| 卡片阴影 | 0 2px 12px rgba(0,0,0,0.08) |
| 卡片hover | 上移 transform translateY(-4px), 阴影加深 |
| 卡片圆角 | 12px |
| 卡片内边距 | 24px |
| 卡片图片 | 顶部全宽图片, border-radius 12px 12px 0 0 |
| 卡片标题 | 18px, 600, margin 16px 0 8px |
| 卡片正文 | 14px, #6B7280, line-height 1.6 |
| 卡片底部操作 | "了解更多"链接, 主色, 14px |

---

## 七、页脚

| 参数 | 值 |
|------|-----|
| 背景 | #1A1A2E 或 主色深 |
| 内边距 | padding 60px 0 30px |
| 结构 | 4列链接（产品/服务/关于/联系）+ 底部版权行 |
| 列标题 | 16px, 600, #FFFFFF |
| 链接 | 14px, rgba(255,255,255,0.7), hover #FFFFFF |
| 链接间距 | margin-bottom 10px |
| 版权行 | 12px, rgba(255,255,255,0.4), border-top 1px rgba(255,255,255,0.1), padding-top 24px |
