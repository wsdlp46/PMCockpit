# 02当前项目

> 进行中的项目放在这里。每个项目一个子目录，目录名即项目名。

## 怎么建一个新项目

本项目目录下有一个 **`_模板项目/`**，它是完整的项目骨架模板。

### 方式 A：复制模板（推荐，结构最完整）

```bash
cd 02当前项目
cp -R _模板项目 你的项目名
```

复制后，把里面所有 `<占位符>` 替换成真实内容。详见 [`_模板项目/README.md`](_模板项目/README.md)。

### 方式 B：让 AI 建

对 AI 说"开始新版本"，AI 会自动建出版本目录骨架。但项目根骨架（00-项目总览、01-项目基线）建议用方式 A 先建好。

## 注意

- 项目名用业务名，不加版本号（版本在迭代目录里管）
- 每个项目独立完整，不与其他项目混目录
- 项目结项后移到 `04历史项目/`
- `_模板项目/` 本身不是真实项目，不要在里面填业务内容（它是模板，复制出去再用）

## 项目目录速查

每个项目的标准结构（详细见 `_模板项目/README.md`）：

```
<项目名>/
├── 00-项目总览.md          ← 仪表盘
├── _index.md               ← AI 索引
├── 01-项目基线/            ← 跨版本资产
├── 02-迭代/                ← 版本目录
├── 03-对外材料/            ← 对外文档
└── _source/、归档/
```
