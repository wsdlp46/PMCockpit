# 02当前项目

> 进行中的项目放在这里。每个项目一个子目录，目录名即项目名。

## 怎么建一个新项目

对 AI 说"开始新版本"，AI 会引导你创建项目结构。或者手动建：

```
02当前项目/
└── <你的项目名>/
    ├── 00-项目总览.md          ← 项目仪表盘
    ├── 01-项目基线/
    │   ├── 关键决策.md          ← 跨版本累积的设计决策
    │   └── 会议记录/
    └── 02-迭代/
        └── _index.md           ← 版本索引
```

## 注意

- 项目名用业务名，不加版本号（版本在迭代目录里管）
- 每个项目独立完整，不与其他项目混目录
- 项目结项后移到 `04历史项目/`

> 这个目录现在为空。建第一个项目时，把本 README 保留或删除均可。
